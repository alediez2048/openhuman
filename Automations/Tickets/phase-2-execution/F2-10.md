# F2-10 Primer: `composio_event` Trigger

**For:** New coding-agent session
**Project:** OpenHuman — Workflows & Automations
**Previous work:** F2-9 shipped the webhook trigger. Composio domain already publishes `DomainEvent::ComposioTriggerReceived`.

---

## What Is This Ticket?

Subscribe to `DomainEvent::ComposioTriggerReceived` and dispatch matching workflows. The Composio domain already routes incoming Composio webhooks into the event bus; F2-10 hooks workflow execution into that fan-out.

### Why It Matters

- Lets users build workflows reacting to any Composio toolkit's events ("when a new GitHub issue is opened in $repo, summarise and send to Slack"). Composio's catalog covers ~250 toolkits, so this trigger unlocks a huge surface without per-integration work.

---

## What This Ticket Must Accomplish

### Goal

- `Trigger::ComposioEvent { toolkit: String, trigger_id: String }` already declared.
- New `workflows::bus::ComposioEventSubscriber` (mirrors F-3's `WorkflowHealthRecomputeSubscriber`):
    - Filters `DomainEvent::ComposioTriggerReceived { toolkit, trigger_id, payload }` against enabled workflows with matching trigger configs.
    - For each match: `executor::dispatch_run(config, workflow_id, TriggerSource::ComposioEvent { payload })`.
    - Multiple workflows can match the same event (fan-out).
- Validator: `toolkit` + `trigger_id` must reference a Composio trigger that exists.
- Honor single-flight (F-9): a second Composio event for an already-running workflow drops + publishes `WorkflowRunSkipped`.

### Deliverables Checklist

#### A. Implementation

- [ ] `workflows::bus::ComposioEventSubscriber` + register at boot.
- [ ] `workflows::store::list_workflows_matching_composio_event(toolkit, trigger_id)` query.
- [ ] Validator arm.

#### B. Tests

- [ ] Publish `ComposioTriggerReceived` → matching workflow dispatches.
- [ ] Disabled workflow doesn't dispatch.
- [ ] Fan-out: 2 workflows matching the same event both dispatch.
- [ ] Single-flight: re-publish while a run is in-flight → second drop + Skipped event.
